Qual a diferença de head e body?

A tag head é onde você deve colocar tags de configuração como title, meta, link e etc. Ela é a responsavel por prover informações gerais sobre o documento. Você pode dar uma lida aqui: https://developer.mozilla.org/pt-BR/docs/Web/HTML/... O body é onde você coloca todo o conteúdo de fato do seu site. Dá uma olhada aqui: https://developer.mozilla.org/pt-BR/docs/Web/HTML/...

Queria saber se a tag body tem auto fechamento.

A tag body necessita de tag de fechamento. Sempre que tivermos uma tag que tenha conteúdo dentro dela, necessitamos utilizar a tag de fechamento. Por mais que algumas tags o navegador consiga fechar sozinho sem a tag de fechamento, isso é uma má prática porque cada navegador trabalha de uma forma e ele pode acabar não interpretando da maneira que queríamos, por isso não devemos deixar o navegador tomar decisões.

Você pode pular quantas linhas quiser. Essas linhas em branco são desconsideradas. Se você está se referindo a espaços do lado esquerdo das tags, isso se chama identação. Fiquei com uma dúvida sobre o DOCTYPE, eu li sobre o que você falou sobre versões antigas do HTML que precisam de uma marcação diferente. Seria o ideal então criar uma página com a marcação do DOCTYPE pro HTML5 e pra umas versões anteriores tipo HTML4 também? ou é desnecessário?

Você sempre vai criar usando a tag do HTML5 <!DOCTYPE html>, o que pode acontecer é vc pegar um site antigo para fazer um suporte, e ele estar usando uma versão antiga do HTML, ai vc pode trazer esse código para o HTML5 mas ai vc precisa atualizar as tags que não são suportadas pelo HTML5. Ou vc só faz a alteração que precisa e deixa o código como esta. Criar um site usando uma versão antiga do HTML não é nada legal, vc vai criar um site totalmente ultrapassado.
