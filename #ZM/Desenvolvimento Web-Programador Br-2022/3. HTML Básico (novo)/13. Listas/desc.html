Dá sim, mas para isso vamos usar o CSS, o CSS é responsável pelas alterações do layout da página, estilo dos textos, cores, tamanhos, posição dos elementos etc. Isso vai ser mostrado nos próximos módulos.

Não entendi a diferença entre uma lista ordenada e uma lista desordenada.

A principal diferença vai ser em sua apresentação dependendo do conjunto de itens que você está usando. Se por exemplo você precisa demonstrar um conjunto de itens sequenciais, como por exemplo, a sequencia dos nomes dos melhores jogadores de um game, você poderia usar a lista ordenada pois teria uma sequencia. Se o conjunto de itens não necessariamente importar a ordem, você pode usar a lista não ordenada.​​​​​​​​​​​​​ ​
