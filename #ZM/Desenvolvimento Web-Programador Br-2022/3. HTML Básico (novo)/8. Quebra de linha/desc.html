É importante conhecer a Tag de quebra de linha para caso você veja em algum código, ou para utilizar em algo muito muito simples, porém durante o curso veremos formas melhores de fazer isso via CSS e nosso objetivo é que depois disso você quase nunca mais use a Tag BR



​DÚVIDAS DOS ALUNOS

Eu fiz a quebra de linha usando a tag "p" e deu certo também. Mediante a essa situação, gostaria de saber se seria errado fazer o que eu fiz ou isso também seria uma alternativa, uma forma de similaridade, assim como entre as tags "del" e "s".

Então, não é uma prática aconselhada não, você verá em módulos posteriores o motivo da tag "p", e algumas além dela, quebrarem a linha. Se você quer a quebra da linha você utiliza a tag "br", se quer um espaçamento, você verá melhor nas aulas de css.

Gosto de usar o <hr> também. mas gostei dessa tag.

Então o hr cria uma linha. O br irá passar para a próxima linha sem a criação dessa "linha" no meio, essa é a diferença entre elas.

Usar várias tags <br> para fazer um espaçamento maior não seria uma prática ruim?

Sim. Com CSS você vai entender melhores formas de fazer isso. Se for uma coisa simples que você usando um <br> poderia resolver, mas no CSS a gente tem a autonomia e liberdade de escolher a quantidade específica de espaços que queremos em certas regiões, nos tirando a carga de encher e sujar nosso código HTML, pois o ideal é deixar o HTML só como esqueleto e alterar tudo com CSS. Para casos pequenos você pode usar as diversas tags de espaços e quebras, fica a seu critério, porém quando começa a acumular torna-se necessário a necessidade de alterar direto no CSS.
