​DÚVIDAS DOS ALUNOS

Preciso usar o Google Drive?​

Você pode usar a Dropbox, o OneDrive, o SharePoint. O que não você não pode fazer é enviar arquivos de códigos como anexos no email.

Posso mandar só o arquivo que está dando problema?​​​​

Não, você deve mandar a pasta do seu projeto. Só desta forma nossa equipe de suporte vai conseguir entender onde está o erro.

Como faço para criar um link compartilhado no Google Drive?​​​

  1. Faça o upload da pasta do seu projeto para o Google Drive
  2. Nesta pasta, clique em “Gerenciar Acesso” (Manage access)
  3. Na janela que abrir, ao invés de “Acesso Restrito” (restricted access), troque para “Qualquer pessoa que tenha o link” (anyone with the link)
  4. Copie o link (Copy the link)
  5. Cole o link no corpo do email que você está preparando para enviar para o nosso suporte
