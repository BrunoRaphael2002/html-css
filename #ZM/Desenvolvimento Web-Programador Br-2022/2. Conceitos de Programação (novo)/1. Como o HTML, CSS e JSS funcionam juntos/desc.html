DÚVIDAS DOS ALUNOS

​Não entendi a parte que diz que o javascript é responsável pela modificação da página depois que ela é carregada.

Quando você clica em um botão, você, usuário, está interagindo com a página, controlando o cursor do mouse até aquele botão e clicando nele, para que esse botão faça alguma cosia. É o JS que manipulará tudo e fará com que algo aconteça. Você não precisa esquentar com isso no momento, tudo será melhor explicado com calma no decorrer dos módulos.

Poderia esclarecer o que é Framework?

O framework é um conjunto de código genérico para resolver determinados problemas. Com base nesse código genérico você consegue criar projetos, facilitando o desenvolvimento de sua aplicação. Ele te ajuda principalmente para otimizar seu tempo de desenvolvimento fazendo com que você não precise criar tudo do zero. Posteriormente você verá melhor isso como tudo isso funciona.
