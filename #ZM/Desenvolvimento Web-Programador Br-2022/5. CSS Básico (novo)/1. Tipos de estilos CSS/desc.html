Como saber a hora certa de usar das 3 opções naquele momento?

​O CSS interno, como vc vai ver em algumas aulas, o Igor vai usar em testes para mostrar algumas funcionalidades, mas ao construir um site vc vai criar um arquivo de CSS externo pra deixar mais organizado. Isso não é uma lei. Vc pode deixar o CSS no HTML (CSS interno) se for pequeno, não vai causar problema algum. 

Já o CSS inline é menos indicado que vc use ele com frequência, porque ele pode acabar atrapalhando o CSS interno ou externo. Vc vai usar ele em casos bem específicos, vamos supor que vc vai enviar um e-mail através de código, para formatar esse e-mail vc vai usar HTML e CSS, nesse caso provavelmente vc vai ter que usar CSS Inline para que o e-mail seja interpretado corretamente.
