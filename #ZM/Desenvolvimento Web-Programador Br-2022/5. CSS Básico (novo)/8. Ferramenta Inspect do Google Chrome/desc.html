Não é perigoso nosso código ser visível para para outros programadores?

​Um site simples normalmente não tem dados sensíveis. Se o seu sistema tem dados sensíveis, ou precisa armazenar algo em banco de dados e etc, isso é feito através do backend (vc verá aqui no curso). Nesse caso, todo o seu código do backend não vai ficar disponível para visualização.

OBS: dados sensíveis são dados que não deveriam ficar expostos, dados importantes como o endereço da sua casa, seu CPF, a conta do seu banco, isso são dados sensíveis.​
