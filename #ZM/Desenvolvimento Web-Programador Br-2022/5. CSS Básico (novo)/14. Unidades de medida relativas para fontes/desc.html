DÚVIDAS DOS ALUNOS

​Quando que eu vou usar o EM ou o REM? eu tbm não entendi em que momento ou porque deveria usar essa medidas.

Nem toda unidade de medida você vai utilizar muito. Normalmente utilizamos mais o px, em, rem, %, vw, vh os demais utilizamos para casos mais específicos.

​A unidade relativa "em" seguirá sempre o valor do font-size do seu pai.

Com relação a utilização da propriedade "em" relacionada ao tamanho dos "px" tenho uma dúvida. Ao invés de fazer todo um cálculo para saber quantos "em" preciso para ter determinado número de pixels, não seria mais fácil e objetivo simplesmente digitar o número desejado de pixels ? Por exemplo:​​​​

.container{ 

font-size: 50px; 

} 

.inside{ 

font-size: 100px; 

width: 200px; 

background: blue; 

}

Me parece mais simples escolher diretamente o número de px para cada propriedade, do que calcular "em".​​

O que acontece é que em é uma unidade relativa usamos esse tipo de unidade para dar responsividade para a página (o Igor irá explicar melhor posteriormente), mas significa uma página que o layout funcione sem quebrar em qualquer tamanho. Vamos dizer que você use 400px numa imagem, ao abrir essa página no celular o layout quebra porque o celular tem menos de 400px. Podemos usar em para deixar todos os tamanhos ligados a fonte, e ai ao invés de alterar os tamanhos dos elementos separados, alteramos apenas a fonte e tudo altera proporcionalmente. Vamos ver como fazer essas alterações de acordo com o tamanho da tela posteriormente.​

​O "Em" só faz referência a font-size? Ou ele pode usar outro referência (elemento) dentro do contêiner?

Você pode usar o em e o rem para controlar o tamanho de outros elementos, mas você precisa ter em mente que fazendo assim, estes elementos vão ficar vinculados ao tamanho da fonte.​
