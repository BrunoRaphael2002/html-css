Eu criei uma div e coloquei um texto qualquer porém quando eu coloquei o texto todo junto sem espaço eles excedeu background somente pela direita, como se eu digitasse um texto em um bloco de notas sem usar o enter para dar uma quebra de linha, e quando eu coloquei um texto com espaço o resultado foi dele vazar por baixo da div, o que causa essa diferença?

Esse é realmente um comportamento padrão, mas tem como mudar isso, usando uma propriedade do CSS word-wrap: break-word, assim vc consegue mudar o comportamento padrão e em vez da palavra exceder a tag ela é quebrada.

Vou deixar um link aqui da documentação da mozilla com mais informações pra vc entender melhor sobre: https://developer.mozilla.org/en-US/docs/Web/CSS/o...​​​

Como eu poderia retirar a barra de rolagem tanto do eixo x, como do y? Pra tipo, colocar um texto rolando por cima de uma imagem, mas sem as barras de rolagem. Obs: a barra de rolagem normal no navegador continua, apenas as barras horizontal e vertical da div iria desaparecer.

Acho que é isso aqui que você está buscando: 

https://www.w3schools.com/howto/howto_css_hide_scr...

/* Hide scrollbar for Chrome, Safari and Opera / 

.example::-webkit-scrollbar { 

display: none; 

} / Hide scrollbar for IE, Edge and Firefox / 

.example { 

-ms-overflow-style: none; / IE and Edge / 

scrollbar-width: none; / Firefox */ 

}

A div apenas aparece quando eu insiro o texto dentro dela. Isso é normal ou algum erro?

A div só é apresentada quando ela tem conteúdo. A menos que você coloque um height e width, ai ela aparecerá.​​​​​​​

Queria saber se essa é a unica maneira de adicionar uma barra de rolagem pra um elemento a parte na página ou se existe algum outro jeito que é ensinado mais a frente no curso.

Atualmente sim, seria a maneira mais viável de adicionar uma barra de rolagem, caso o elemento limitado vaze de suas dimensões.
