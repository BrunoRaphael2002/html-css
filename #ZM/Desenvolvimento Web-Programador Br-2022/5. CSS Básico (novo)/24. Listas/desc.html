No list-style-image, existe um tamanho de imagem padrão para dar certo no site?

Então, na verdade não há, o list style image irá pegar o tamanho original da imagem, e não sendo possível redimensionar, então o ideal é que utilize ícones, você pode baixar alguns aqui: https://icons8.com.br/icons 

Mais para a frente do curso você vai entender como fazer isso de outra forma.​​​

​Quando usamos a tag list-style-image existe algum tipo de parâmetro relacionado ao tamanho ou tipo de imagem a ser adicionada? Ou ao adicionar a imagem todas ficaram com o mesmo tamanho por padrão?

Para podermos alterar a imagem como quisermos temos que fazer um pouco diferente, ai é necessário o uso de CSS já, apenas adiantando o que iremos fazer é tirar o list style e colocar uma imagem junto à lista.​​

​Como eu poderia criar uma linha vertical entre o ícone e a lista?

Uma forma simples seria iniciar o texto com a barra vertical do teclado "|", pode fazer a imagem da lista sendo um elemento diferente e colocar uma borda na lateral direita dela, ou na lateral esquerda da lista e tirar o estilo da lista, pode fazer serem 3 elementos, imagem, linha, lista.

​Como consigo colocar sublistas? tipo tem o item 1, 2, 3, ai tem o 3.1, 3.2, 3.3, mais recuados?

Então não vai ser possível fazer isso de forma direta só com html, seria necessário implementar uma lógica com javascript. O que você pode fazer é usar números primeiro, depois letras para representar o subitem, ou então fazer uma lista não ordenada dentro de uma ordenada. 

Vou deixar um link sobre o assunto https://developer.mozilla.org/pt-BR/docs/Web/HTML/...

​Visivelmente se vê diferença entre a tag ol e ul. mas no caso se estilizar não faz diferença (não pelo menos visual). A dúvida é: para o navegador isso tem alguma diferença ou não?

Não faz diferença, porém é uma boa prática usar ul caso não vá numerar a lista.

​Tem como mudar a fonte da lista?

Tem sim, basta você colocar li { font-family: nome_da_familia;} 

Existem várias familias de fonte, dá uma olhadinha aqui: https://www.w3.org/Style/Examples/007/fonts.pt_BR....
