​Qual a diferença entre o font-weight e a tag strong? recomenda usar uma ou outra?​

Então, a tag "strong" tem um significado semântico, e quando você coloca, significa que você está dando importância para certas palavras, isso pode resultar em um melhor ranqueamento do seu site. O font-weight é apenas uma estilização visual. 

Dá uma olhada aqui: https://developer.mozilla.org/pt-BR/docs/Web/HTML/...

Só completando a resposta, depende muito do resultado que está querendo ter, caso seja só visual, você pode usar o font-weight ou até mesmo a tag "b".

​Quais as diferenças de codar negrito ou italico ou caixa alta no html e, estilizar negrito ou italico caixa alta etc.. no css?​

Depende muito da forma que está fazendo no html. Se está utilizando o strong, por exemplo, ela daria um sentido semântico dizendo que aquele texto é muito importante. O mesmo acontece com a tag em, ela tem um sentido semântico. 

Dá uma olhadinha aqui: https://pt.stackoverflow.com/questions/32289/qual-...

Sobre estilizar esses mesmos efeitos com CSS, é só para estilo mesmo, ou seja, apenas um efeito visual.​

​Eu posso escolher a fonte usando tanto a "font-style" quanto a "font-family"?

Há uma diferença entre ambos, font style muda o estilo do elemento... ex: colocando em italico, mas o font family muda a famila da palarava... você pode olhar exemplos melhores aqui: https://www.w3schools.com/cssref/pr_font_font-styl... e https://www.w3schools.com/cssref/pr_font_font-fami...
