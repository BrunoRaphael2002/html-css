Fala Pessoal,

Depois de uma atualização o módulo node-fetch deixou de funcionar com o require.

Então em vez de usar o "node-fetch" podemos usar o modulo "cross-fetch" sem maiores alterações.

O que muda:


09:20

npm install node-fetch

npm install cross-fetch


09:35

const fetch = require('node-fetch')

const fetch = require('cross-fetch')
