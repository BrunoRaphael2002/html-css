O que são Frameworks?


Frameworks:


O JavaScript é a principal linguagem de programação do mundo. Por todos os lados da web que você observar existe algum site ou aplicação utilizando frameworks JavaScripts e bibliotecas da linguagem.

O framework traduzindo a grosso modo,para o portugues significa estrutura. É um conjunto de codigos prontos que são reultilizados como o intuito de evitar a peda de tempo programando a estrutura de um site do zero.

Ele serve para auxiliar o Programador ou programadora na hora de construir uma aplicação web.

Por exemplo na hora de desenvolver uma pagina de vendas para uma empresa de ou um E-comerce,o Que é mais vantajoso ?

O que é uma biblioteca de JavaScript?

A biblioteca de JavaScript é um "Trecho de codigo reutilizavel" que serve de base para desenvolver outros codigos, com a principal ideia de desenvolver codigos sem perder tempo escrevendo codigos que outro programador já escreveu.

Principais Frameworks

React.Js

O react foi desenvolvido em 2013 pelo Facebook. É uma biblioteca de codigo aberto e tem o apoio do Facebook.o React é usado para ajudar a criar aplicações na web em pequena ou larga escala, principalmente em interfaces interativas.

Angular.Js

É um frameWork mais importantes. É um framework Front-end de codigo aberto baseado em JavaScript Ele utiliza HTML como uma linguagem de modelo Dinamico.

bootstrap

O Bootstrap é um dos principais frameworks JavaScript web de codigo aberto do mercado. É capaz de oferecer padrões para o desenvolvimento HTML, CSS e JavaScript.

Foi desenvolvido em 2010 e lançado oficialmente em 2011 pelos profissionais do Twitter, Mark Otto e Jacob Thornton.

O Bootstrap facilita a criação de Layouts, sendo possível criar designs responsivos.
